# Data

This folder is intentionally empty in the repo — `fraudTrain.csv` (335 MB) and
`fraudTest.csv` (144 MB) exceed GitHub's file size limits.

**Download the dataset here:**
Kaggle — [Credit Card Transactions Fraud Detection Dataset](https://www.kaggle.com/datasets/kartik2112/fraud-detection)

Place both CSVs in this folder before running the scripts:
```
data/
├── fraudTrain.csv
└── fraudTest.csv
```
